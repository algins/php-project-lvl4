### Requirements:
* Composer
* Git
* Make
* Node.js
* PHP ^8.0
* SQLite

### Setup:
```sh
$ git clone https://github.com/algins/php-project-lvl4.git
$ cd php-project-lvl4
$ make setup
```

### Run:
```sh
$ make start # Open http://localhost:8000
```
